AI Document Summarizer

This project demonstrates an automated workflow using Zapier/n8n and AI. It ingests a PDF document, extracts the content, summarizes the key insights with OpenAI, logs the summary in Google Sheets, and sends the results through Slack or email.

Features

  • Uploads and ingests a PDF document
  • Extracts the text for processing
  • Summarizes key insights using OpenAI
  • Logs the summary into Google Sheets
  • Sends results via Slack or email

Stack

n8n or Zapier • OpenAI API • Google Sheets • Slack • PDF parsing
