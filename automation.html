Automation Workflow Project

This project demonstrates an automated workflow using Zapier/n8n and AI. It takes a form submission, logs it in Google Sheets, sends a Slack alert, and generates a personalized email draft with OpenAI.

Features

  • Form submission capture
  • Google Sheets integration
  • Slack notification
  • AI-generated follow-up email
