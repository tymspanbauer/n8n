Automation Workflow

This project demonstrates an automated workflow using Zapier/n8n and AI. It takes a form submission, logs it in Google Sheets, sends a Slack alert, and generates a personalized email draft with OpenAI.

Features

  • Captures form submissions automatically
  • Writes rows to Google Sheets
  • Sends Slack notifications to a channel
  • Generates a tailored email draft with OpenAI
  • Easily extendable to CRMs or ticketing tools

Stack

Zapier or n8n • Google Sheets • Slack • OpenAI API
